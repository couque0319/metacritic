# 게임 리뷰 웹사이트 프로젝트

게임 정보를 표시하고 사용자가 리뷰를 작성할 수 있는 웹 애플리케이션입니다.

## 프로젝트 구조

```
project-root/
├── frontend/       # React 프론트엔드 코드
│   ├── src/
│   │   ├── components/
│   │   ├── pages/
│   │   └── ...
│   └── public/
└── backend/        # PHP 백엔드 코드
    ├── api/        # API 엔드포인트
    ├── config/     # 설정 파일
    └── db/         # 데이터베이스 관련
```

## 기능

- 게임 목록 표시
- 게임 상세 정보 보기
- 리뷰 작성 및 조회
- 사용자 회원가입 및 로그인

## 기술 스택

### 프론트엔드
- React
- React Router
- CSS

### 백엔드
- PHP
- MySQL

## 설치 방법

1. 리포지토리 클론
```cmd
git clone https://github.com/couque0319/reactproject.git .
```

2. 백엔드 설정
- MySQL 데이터베이스 

3. 프론트엔드 설정
```bash
cd frontend
npm install
npm start
```

4. `.htaccess` 파일로 라우팅 설정
